# 方法写作指南

## 目标
方法部分的核心任务不是展示术语，而是让读者清楚知道：
1. 研究对象是什么
2. 研究材料或数据从哪里来
3. 研究是如何开展的
4. 使用了什么分析方法或工具
5. 为什么这些方法足以回答研究问题

在中文图书情报论文中，方法部分既要说明研究过程，也要体现研究设计的合理性与可解释性。

## 基本原则
1. 方法部分必须服务于研究问题，不能脱离研究目标单独堆砌技术名词。
2. 应交代研究对象、样本范围、数据来源、时间区间、筛选标准。
3. 应说明分析步骤，而不是只罗列工具名称。
4. 应说明方法选择依据，回答“为什么用这种方法”。
5. 如果存在编码、问卷、指标构建、参数设置等关键环节，应说明其判断依据。
6. 如果研究存在边界或局限，应适度提示。
7. 方法写作应以清楚、可复核、可理解为优先，而不是追求复杂化表达。

## 通用结构
### 模板 A：对象—数据—方法—步骤—可靠性
适用于多数实证与分析型论文。

- 对象：本文研究的对象、场景、机构、用户或文本材料是什么
- 数据：样本来源、样本规模、时间范围、筛选规则
- 方法：使用的研究方法、分析框架、模型或工具
- 步骤：研究具体如何展开
- 可靠性：编码一致性、量表信效度、数据清洗、结果校验等

### 模板 B：研究设计—数据处理—分析路径
适用于文献计量、知识图谱、文本分析和政策分析类论文。

- 研究设计：研究目标、研究问题、总体设计思路
- 数据处理：检索式、筛选条件、去重规则、清洗方法
- 分析路径：采用何种指标、模型、维度或框架完成分析

## 分类型写法
### 一、文献计量 / 知识图谱类论文
建议写清：
- 数据库来源
- 检索时间与检索式
- 纳入与剔除标准
- 文献数量与处理过程
- 使用的软件或分析方法
- 分析维度，如作者、机构、关键词、主题演化、共现网络等

推荐句式：
- 以某数据库收录的文献为数据来源，检索时间范围为……
- 经过去重、清洗和人工筛选后，最终获得……篇样本文献。
- 运用……软件 / 方法，从……维度对样本进行分析。

必须避免：
- 只写使用了 CiteSpace、VOSviewer 等工具，不写样本如何得到
- 不交代检索范围、字段或清洗过程
- 把工具名称当成方法说明的全部内容

### 二、问卷 / 访谈 / 用户行为研究类论文
建议写清：
- 研究对象与抽样方式
- 问卷 / 访谈提纲来源
- 变量或维度设计
- 数据回收情况
- 分析模型或统计方法
- 信度、效度或编码一致性

推荐句式：
- 研究以……为对象，采用……方式发放问卷 / 开展访谈。
- 问卷条目主要参考……并结合……情境进行修订。
- 运用……方法对数据进行分析，以检验……

必须避免：
- 不说明样本来源与有效样本数
- 不说明量表或访谈提纲依据
- 只有统计名词，没有研究步骤

### 三、政策 / 文本 / 内容分析类论文
建议写清：
- 材料来源与选取标准
- 分析单元与分析维度
- 编码框架或分类标准
- 分析过程与判断依据
- 必要时说明例证选择逻辑

推荐句式：
- 本文选取……作为分析材料，材料获取时间为……
- 在综合既有研究与研究对象特征的基础上，构建……分析框架。
- 依据……对文本进行编码、归类与比较分析。

必须避免：
- 只说“采用内容分析法”，不写分析框架
- 不交代材料如何选取
- 不说明分类标准从何而来

### 四、案例研究类论文
建议写清：
- 为什么选择该案例
- 案例具有什么代表性或特殊性
- 材料来源有哪些
- 分析围绕哪些问题展开
- 结论适用边界是什么

必须避免：
- 把案例描述当成方法说明
- 不交代案例选择理由
- 从单一案例直接推出过强结论

## 方法部分常见问题
### 问题 1：只报方法名，不写研究过程
处理：增补对象、材料、步骤与参数 / 标准信息。

### 问题 2：方法与研究问题脱节
处理：检查方法是否足以回答问题；必要时弱化研究问题或补充分析步骤。

### 问题 3：研究设计顺序混乱
处理：按“对象 / 数据—处理—分析—校验”顺序重组。

### 问题 4：关键判断依据缺失
处理：将所有会影响结论可信度的关键环节补写出来。

### 问题 5：方法过度技术化，脱离论文读者
处理：用“本文为何这样做、这样做能回答什么问题”的表达重新组织方法说明。

## 方法改写时的默认输出要求
当处理方法部分时，应输出：
1. 当前方法写作中的主要问题
2. 推荐的结构顺序
3. 改写后的方法部分
4. 仍需补充的样本、检索式、编码规则、变量定义或校验信息

## 好方法部分的判断标准
一篇较好的方法部分通常满足：
- 能清楚看出研究对象、样本和数据来源
- 能看出研究是如何一步步展开的
- 关键判断标准公开透明
- 方法与研究问题相互对应
- 读者能够理解结论为何成立及其边界何在
