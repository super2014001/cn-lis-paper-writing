# 结果与讨论写作指南

## 目标
结果与讨论部分的核心任务，是把“研究发现了什么”与“这些发现说明了什么”区分开并衔接起来。

- 结果：回答看到了什么、得到了什么
- 讨论：回答为什么会这样、与既有研究关系如何、对实践有什么启示

在中文图书情报论文中，这一部分尤其要避免“只复述结果”或“脱离结果直接拔高”。

## 基本原则
1. 结果与讨论可以分开写，也可以合并写，但功能必须区分清楚。
2. 结果应围绕研究问题展开，按照问题顺序或分析维度展开。
3. 讨论应建立在结果基础上，不能脱离结果空谈意义。
4. 实践启示应尽量具体到图书馆服务、信息组织、资源建设、平台治理、用户支持或公共文化服务场景。
5. 不应把“现象描述”直接等同于“机制解释”。
6. 不应把局部发现无限上升为普遍结论。

## 常用结构
### 模板 A：结果—解释—比较—启示
适用于大多数论文。

- 结果：报告核心发现
- 解释：分析发现形成的原因或逻辑
- 比较：与既有研究或不同场景进行对照
- 启示：提出理论或实践上的意义

### 模板 B：按研究问题逐项展开
适用于有明确研究问题或假设的论文。

对于每一个研究问题，分别写：
1. 对应结果
2. 结果解释
3. 与前人研究关系
4. 对实践的启示

### 模板 C：按主题 / 维度展开
适用于文献计量、内容分析、案例研究类论文。

例如：
- 主题一：结构特征
- 主题二：演化趋势
- 主题三：主要问题
- 主题四：实践建议

## 结果部分写法
### 结果部分应该写什么
1. 与研究问题直接相关的核心发现
2. 样本呈现出的主要特征、差异、关系或趋势
3. 图表所支持的关键信息
4. 必要时对发现进行分层或分类概括

### 结果部分不应该写什么
- 大段重复方法过程
- 与研究问题无关的枝节数据
- 还没有证据支持的原因推断
- 纯粹重复图表标题和数字

### 推荐句式
- 研究结果表明，……
- 从……维度看，样本主要呈现出……特征。
- 进一步分析发现，……之间存在……关系。
- 不同类型样本在……方面表现出明显差异。

## 讨论部分写法
### 讨论部分应该写什么
1. 对结果进行解释
2. 指出结果与既有研究的一致或差异之处
3. 说明研究发现对学科理解有什么补充
4. 提出对图书馆实践、信息服务或政策治理的启示
5. 必要时提示研究边界

### 讨论部分不应该写什么
- 脱离结果空谈趋势
- 只重复“这很重要”
- 只说“具有实践意义”但不说明落点
- 用单一结果推出过于宏大的结论

### 推荐句式
- 这一结果表明，……
- 与既有研究相比，本文发现更强调……
- 出现这一现象的可能原因在于……
- 从实践层面看，该发现提示图书馆 / 信息服务机构应……

## 分类型写法
### 一、文献计量 / 知识图谱类论文
结果部分建议写：
- 年度发文趋势
- 核心作者 / 机构分布
- 关键词聚类或主题结构
- 热点演化与前沿趋势

讨论部分建议写：
- 研究热点为何形成
- 学科结构有何变化
- 哪些方向值得进一步关注
- 对学科布局、资源建设、科研支持有什么启示

### 二、问卷 / 访谈 / 用户行为研究类论文
结果部分建议写：
- 样本描述
- 变量关系
- 显著影响因素
- 类型划分或机制结构

讨论部分建议写：
- 为什么会出现这些关系
- 与既有研究是否一致
- 对用户服务、平台设计、信息干预有什么启示

### 三、政策 / 文本 / 内容分析类论文
结果部分建议写：
- 主题、类型、框架、结构或问题特征
- 不同政策 / 文本 / 案例之间的共性与差异

讨论部分建议写：
- 这些结构特征反映了什么治理逻辑或制度特征
- 现有实践的短板在哪里
- 对制度优化、政策完善、服务改进有何启示

### 四、案例研究类论文
结果部分建议写：
- 案例做法、机制、过程、成效与问题

讨论部分建议写：
- 哪些经验具有可迁移性
- 哪些因素依赖特定情境
- 对类似机构或场景有何借鉴意义

## 结果与讨论常见问题
### 问题 1：结果与讨论混在一起但层次不清
处理：用“结果是什么—说明了什么—启示是什么”的顺序重组段落。

### 问题 2：只描述现象，不解释原因
处理：在每个核心结果后补充原因分析、机制解释或与既有研究的比较。

### 问题 3：讨论脱离结果
处理：检查每条讨论是否都能回指到前文具体结果或证据。

### 问题 4：实践启示空泛
处理：明确指出针对谁、改进什么、在什么场景下实施。

### 问题 5：结论提前写完，讨论部分失去功能
处理：讨论聚焦“解释与延伸”，结论聚焦“总括与收束”。

## 结果与讨论改写时的默认输出要求
当处理结果与讨论时，应输出：
1. 当前部分存在的问题
2. 推荐的展开顺序
3. 改写后的结果与讨论文本
4. 每一段分别属于“结果”还是“讨论”
5. 仍需补充的证据、比较文献或实践落点

## 好的结果与讨论部分的判断标准
一篇较好的结果与讨论通常满足：
- 能清楚区分“发现”与“解释”
- 每项讨论都有结果支撑
- 能回应研究问题
- 能与既有研究形成对话
- 能提出具体而不过度拔高的实践启示
